WP Post Duplicator is a best way to make an exact duplicate of a selected Post/Page/Custom Post along with categories, taxonomies & meta fields.
 
== Description ==

WP Post Duplicator provides you an direct link on all Post / Page / Custom Post to make the exact duplicate for that along with categories / custom taxonomies, tags / custom tags and meta fields with full compatibility with Option Tree Meta fields.

= Features =

* Clone posts, pages and custom post types
* Functionality to clone post along with their categories, tags, custom taxonomies, custom tags and all their meta fields
* Provides direct link with post / page listing to make that duplicate
* Cloning multiple posts / Bulk post duplicator
* Option Tree Meta fields compatible
* Light weight plugin


== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ folder of your WordPress installation.  
* Activate the plugin.   
* You will see a  direct link with post / page listing to make that duplicate. 

Enjoy it......
